# Overview
A minimal Unix shell built in C for academic use. 
Supports external commands, I/O redirection, pipes, 
background jobs, built-ins (cd, exit), signal handling, 
and command logging. Designed for Valgrind clean and automated testing.

## Build Instructions
make

## Run Instructions
./myshell
