# myshell-project
**Authors** Manheet Kaur Virk, Jayla Porter, Dapinderdeep Kaur
**Course** CS 126-001 Systems Programming in C

##Overview 
A minimal Unix shell built in C for academic use. Supports external commands, I/O redirection, pipes, background jobs, built-ins (cd, exit), signal handling, and command logging. Designed for Valgrind cleanliness and automated testing.
